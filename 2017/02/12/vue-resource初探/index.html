vue-resource初探

  • vue-resource

简介

vue-resource是用来发送ajax请求或者解决jsonp跨域问题，既可以在全局Vue中使用（即Vue.$http）也可以在vue实例中使用（即this.$http）

安装

npm $ npm install vue-resource
cdn <script src="https://cdn.jsdelivr.net/vue.resource/1.0.3/vue-resource.min.js"></script>

基本用法
